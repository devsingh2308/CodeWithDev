Solving: Python Not Found, Run Without Arguments

The error message "Python not found; run without arguments to install from the Microsoft Store" is quite common when Python isn't properly added to your system's PATH variable on Windows.

This message typically appears when you try to run `python` or `python3` from the command prompt, but Windows can’t locate the Python executable. If Python is installed but not in PATH, the system doesn’t know how to execute the command.

To resolve this, the most straightforward fix is to reinstall Python and check the option that says “Add Python to PATH” during setup. Alternatively, you can manually add Python’s installation directory to your environment variables.

You can check whether Python is installed by navigating to the installation directory, typically: C:\Users\YourName\AppData\Local\Programs\Python\Python311

Then, add this path to your system's PATH environment variable. Restart your terminal and try running `python` again.

If the issue persists, ensure there are no conflicting installations or incorrect symbolic links interfering with the command.
