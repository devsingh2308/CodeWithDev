C++ Random Numbers Guide

Generating random numbers in C++ is an essential skill for many types of applications, including games, simulations, and randomized testing. C++ provides both traditional methods using rand() and modern approaches with the C++11 random library.

The traditional way involves calling rand() from <cstdlib> and seeding it with srand():
srand(time(0)); int num = rand();
This works but lacks advanced features like distribution control and reproducibility.

For more robust and reliable random number generation, C++11 introduced the <random> header, which includes engines (like mt19937) and distributions (like uniform_int_distribution).

Example:
std::random_device rd;
std::mt19937 gen(rd());
std::uniform_int_distribution<> dis(1, 10);
int random_number = dis(gen);

This modern approach allows you to generate random numbers with full control over distributions such as uniform, normal, or binomial, making it ideal for scientific computing or AI simulations.

If reproducibility is important (e.g., in testing), use a fixed seed instead of random_device, like std::mt19937 gen(1234);. This guarantees the same sequence of numbers on each run.
